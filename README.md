## Description
- Figured out how to layer correctly as to go from white (top center) to black (at the edges).
- Now has 5 major parameters:
	- Desired size. Specifies the approximate width in pixels of the final tree. 
	- Rescale for blur. The amount to upscale the resolution. This has a large effect on the blurring operation.
	- Blur amount. 
	- Threshold value. The value which pixels less than this value get discarded after the blur. 
	- Number of layers.
